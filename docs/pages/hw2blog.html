My challenges with this assingment included getting the navbar to be a certain distance away from the other sibling elements, getting a perfect match with the borders/colors/shadows, and making the twit-author class set correctly when the twit-container is flexed.

I worked through the navbar issues by using a calc() to determine the amount to transform along the X-axis. I used a color picker to get a perfect match with the colors and played around with the border/shadow through trial and error to get a match there. I used max-width/height and flex-basis to make the twit author fit in the container. I had to google how to focus a search bar, and how to center a flexbox.

I learned how to use CSS to mockup a webpage given the html. I also learned how to use CSS to make calculations, and make superb box shadows. I already knew how to approach the problem, but the actual fine implementaton ideas needed brushing up. The most helpful resource in this assignment process was the TA's who've held office hours. I had a really specific issue--they were able to help me work through it without giving me the answer!
