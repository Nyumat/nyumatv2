One really specific challenge I faced on the creation of this assignment was the fading the navigation bar on the scroll of the web page. The way I fixed this was creating an window.onscroll event which is a callback function that checks the distance from the top of the page on scroll. If it is within a selected range from the top or bottom, I wrote a seperate function which takes in a class and fades in/out that element.










In this assignment, I learned how to deploy a website through the OSU engineering servers, how to create a linear gradient background, and, how to maniuplate elements using jquery and the DOM. I already knew most of the content, but I wanted to try doing something I didn't which is why I added so many animations to this. This web app well end up being the hub for all my assignments so I wanted to make it as modular as possible.
