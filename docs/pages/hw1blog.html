My challenges with this assignment included but weren't limited to: fading the nav bar when it 'collides' with another element, not showing the loading screen on refresh, and, implementing a linear gradient. I solved the fading of the nav bar(s) by utilizing the onscroll event listener and scrollY property to determine if it was within limits to be faded out. I solved the loading screen problem by strapping the logic into a Cookie on the web page which expires after 3 minutes. I implemented the linear gradient by using modern css.




In this first homework, I learned how to use use vanialla javascript to manipulate the DOM, how to upload a single page application to the ENGR servers, and how to incorparate a splash screen.
